# 版本管理

```xml
<properties>
  <org.springframework-version>3.2.4.RELEASE</org.springframework-version>
</properties>

<dependencies>
  <dependency>
    <groupId>org.springframework</groupId>
    <artifactId>spring-core</artifactId>
    <version>${org.springframework-version}</version>
  </dependency>
</dependencies>
```
