# cron

- 表达式格式

```shell
*    *    *    *    *    *
┬    ┬    ┬    ┬    ┬    ┬
│    │    │    │    │    │
│    │    │    │    │    └ day of week (0 - 7) (0 or 7 is Sun)
│    │    │    │    └───── month (1 - 12)
│    │    │    └────────── day of month (1 - 31)
│    │    └─────────────── hour (0 - 23)
│    └──────────────────── minute (0 - 59)
└───────────────────────── second (0 - 59, OPTIONAL)
123456789
```

```shell
#每份钟的第42秒的时候,来运行这个定时任务
42 * * * * *

#每小时的第42分钟来运行这个任务
42 * * * *
#注意想在每小时多少分钟来运行,不能写成下面这样,不然会42分钟的每秒钟都在运行
* 42 * * * *

#我想2分钟执行一次定时任务表达式如下
*/2 * * * *

#在每天的11点05分11秒运行这个任务
11 05 11 * * *

#每天11点06分运行这个任务
06 11 * * *
```
